# WebShopX Issues & Feedback

[WebShopX README](./README-WebShopX.md) | [WebshopX Wiki](https://github.com/Cc-Cece/WebShopX-Issues/wiki)

**WebShopX** 是一款面向 Paper / Spigot 服务器的专业 Web 商店插件。它将官方商城、玩家市场、钱包、订单系统和后台管理深度集成，提供一站式的商业化解决方案。

> 📢 **当前状态：Beta 公测中**
> 核心代码目前为**非公开**状态，本项目仓库仅用于 **问题追踪**、**建议反馈** 以及 **开发者/专业用户入组申请**。

---

## ✨ 项目特性
* **双系统集成**：官方商城 (B2C) 与 玩家市场 (C2C) 共用一套账号、钱包与订单逻辑。
* **Web 优先**：内置响应式玩家端与管理后台，支持内置 HTTP 模式或外部 Nginx/CDN 托管。
* **多币体系**：原生支持 ShopCoin / GameCoin 双币，并可通过 Vault 对接现有经济插件。
* **闭环管理**：涵盖自动发货、手动领取 `/ws claim`、退款、兑换码及团购券系统。
* **管理审计**：细粒度管理员权限、用户能力重制（密码/解绑/强制下线）及审计日志。

## 🛠 反馈指南
我们欢迎所有公测用户参与完善项目：
1. **搜索旧 Issue**：提交前请先确认问题是否已被他人反馈。
2. **使用模板**：请务必根据需求选择对应的模板。
3. **提供环境信息**：报错类 Issue 必须附带 **Java 21** 环境说明及服务器完整日志。

## 👨‍💻 开发者与重度用户入组 
WebShopX 计划在未来逐步开源。目前我们开放少量名额邀请外部人员参与协作，但为了确保开发效率，**普通用户（非代码贡献或深度测试者）请勿申请**。

### 🚫 准入说明
* **普通用户**：请通过 Issue 报告 Bug 或在社区群组交流。
* **重度用户**：指具备技术背景，能进行多次测试、协助复现复杂逻辑或完善文档的人士。
* **开发者**：需熟悉 Java，拥有 Spigot/Paper API 开发经验，并愿意和计划协作开发WebshopX。

### ✅ 申请方式
若你符合条件并希望参与，请提交 Issue 并选择 **"Developer/Pro-User Application"** 模板。
